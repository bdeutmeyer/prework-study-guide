HTML

The head element contains metadata about the webpage.


Some child elements you could see within the head element are:

  • The meta element: defines metadata like character set, description, keywords, author, viewport, etc.
  • The title element: contains the title of the website (reqired), which appears in the browser tab
  • The style element: defines the default CSS styles for the website
  • The base element: defines the base URL for the website

The body element contains the elements that will be visible to the user, like headings, paragraphs, images, and links.

Some child elements you could see within the body element are:

  • The header element: this could contain the h1 heading for the webpage, a banner image/logo, etc.
  • The main element: this would contain the main webpage structure, like section headings, paragraphs, lists, media, and links.
  • The footer element: this could contain info about the author, copyright details, a sitemap, navigation links, etc.

CSS

3 ways to add CSS

  • Inline style
  • Internal style sheet
  • External style sheet (preferred)

Parts of a CSS ruleset

  • Selector
  • Declaration
    • Property
    • Value

CSS Box Model

  • Margin: space around the outside of the element
  • Border: optional visible border around the edge of the element
  • Padding: space between the border/outside edge of the element and the content within
